# Challenge 02 - PWA y Service Workers
**Clase de Desarrollo de Software para Plataformas Móviles**  
**Autor:** Isaac Piedrahita

Este proyecto transforma una aplicación estándar de React en una Aplicación Web Progresiva (PWA). Se utiliza una estrategia híbrida de caché mediante Service Workers para optimizar el rendimiento y la disponibilidad offline.

## Características Implementadas
- Se añadió una imagen al componente padre de la aplicación.
- Se implementó un ícono personalizado para la PWA.
- Se transformó la aplicación en una PWA utilizando una Estrategia Híbrida (Hybrid Strategy) para el Service Worker.

## Guía de Instalación en Dispositivos Móviles

### Android (Chrome)
1. Abre el enlace de la aplicación en Google Chrome.
2. Toca el **menú de tres puntos** en la esquina superior derecha.
3. Selecciona **Agregar a la pantalla principal** (Add to Home screen) o **Instalar aplicación** (Install app).
4. Confirma la acción cuando aparezca el aviso en pantalla. La aplicación aparecerá en tu pantalla de inicio.

### iOS (Safari)
1. Abre el enlace de la aplicación en Safari.
2. Toca el botón de **Compartir** (el ícono del cuadrado con una flecha hacia arriba) en la parte inferior de la pantalla.
3. Desliza el menú hacia abajo y selecciona **Agregar a inicio** (Add to Home Screen).
4. Confirma tocando **Agregar** en la esquina superior derecha para que la app aparezca en tu pantalla de inicio.
